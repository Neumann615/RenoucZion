# 提交类型

## 主要提交类型

| 类型         | 说明          | 示例                               |
| ------------ | ------------- | ---------------------------------- |
| feat/feature | 新增功能/特性 | feat(user): 添加用户个人中心       |
| fix          | ​修复Bug/错误 | fix(payment): 修复支付金额计算错误 |
| docs         | 文档更新      | docs(readme): 更新项目安装说明     |
| style        | 代码样式/格式 | style: 修复代码缩进问题            |
| refactor     | ​代码重构     | refactor(user): 重构用户服务类结构 |
| pref         | 性能优化      | perf(db): 优化SQL查询性能          |
| test         | 测试相关      | test(login): 添加登录功能单元测试  |
| build        | 构建系统变更  | build: 升级webpack到v5版本         |
| ci           | CI/CD配置     | ci: 添加GitHub Actions工作流       |
| chore        | 杂项/工具变更 | chore: 更新依赖包版本              |
| revert       | 回滚提交      | revert: 撤销错误的用户删除功能     |


## 版本发布相关

| 类型    | 说明      | 示例                       |
| ------- | --------- | -------------------------- |
| release | 版本发布  | release: v2.1.0发布        |
| version | ​版本更新 | version: 升级版本号为1.3.0 |
| tag     | 标签操作  | tag: 添加v2.0.0标签        |

## 前端相关

| 类型   | 说明         | 示例                             |
| ------ | ------------ | -------------------------------- |
| ui     | 用户界面变更 | ui(login): 优化登录页面布局      |
| ux     | 用户体验优化 | ux(search): 优化搜索结果展示效果 |
| design | ​设计/视觉   | design(theme): 更新主题色        |
| layout | 布局调整     | layout(header): 调整导航栏布局   |

## 移动端相关

| 类型    | 说明            | 示例                           |
| ------- | --------------- | ------------------------------ |
| ios     | IOS平台特定     | ios: 修复iOS键盘遮挡问题       |
| android | Android平台特定 | android: 修复Android返回键问题 |
| mobile  | ​移动端通用     | mobile: 优化移动端触摸体验     |

## 数据相关

| 类型      | 说明       | 示例                            |
| --------- | ---------- | ------------------------------- |
| database  | 数据库变更 | database: 新增用户积分表        |
| db        | 数据库操作 | db: 添加数据迁移脚本            |
| migration | ​数据迁移  | migration: 用户表添加手机号字段 |
| seed      | 种子数据   | seed: 添加初始管理员数据        |

## 配置相关

| 类型   | 说明      | 示例                       |
| ------ | --------- | -------------------------- |
| config | 配置文件  | config: 更新数据库连接配置 |
| env    | 环境配置  | env: 添加开发环境配置      |
| deploy | ​部署配置 | deploy: 更新Docker部署脚本 |

## 运维相关

| 类型    | 说明      | 示例                      |
| ------- | --------- | ------------------------- |
| ops     | 运维相关  | ops: 更新nginx配置        |
| infra   | 基础设施  | infra: 更新Terraform配置  |
| monitor | ​监控相关 | monitor: 添加应用性能监控 |

## 包管理

| 类型    | 说明        | 示例                     |
| ------- | ----------- | ------------------------ |
| package | 包管理      | package: 更新npm依赖版本 |
| deps    | 依赖更新    | deps: 升级React到18版本  |
| vendor  | ​依赖库更新 | vendor: 更新第三方库版本 |
